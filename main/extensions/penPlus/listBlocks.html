List Blocks

If you are looking to render individual triangles. Check out Triangle Blocks.

These blocks are for rendering triangles using lists. Note that static meshes are better for rendering using this method.


How do I use these?

First things first you will want to create a list.

Once your list is created you should see the list appear in the menu

Now here come the actual blocks


"Define Triangle"

These two blocks have the same purpose. But They have one unique difference.
one has inputs for UV values. If you want to learn about those go read the Triangle Blocks page
Here is a video example of using the block



"Set [attribute] of point [point] to [value] in [definition]"

This block allows you to edit the attributes of a triangle definition. These attributes are the exact same as the ones on a normal triangle.
They are even stackable!




"Draw solid triangles from list [list]"

This block simply draws the defined triangles in a list.


"Draw textured triangles from list [list] using [costume]"

This block draws textured triangles from a list.
If the list is missing UV info the block will not draw any triangles!


"Draw textured triangles from list [list] using [shader]"

This block draws triangles from a list using the desired shader.
This even includes custom attributes added by the programmer! Like Normals!
