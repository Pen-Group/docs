Control Blocks

"if [condition] then" and "if [condition] then else"

Trigger the inner script if the condition is true.
If there is an else it will trigger that if the condition is false.

"repeat" and "loop index"

Repeat will loop the code for a desired amount of times.
Loop index will allow you to get the index of the current loop.

"break"

Allows you to stop a loop at any point.

"continue"

Allows you to stop a loop at any point, and continue it at the next index.

"discard"

Allows you to remove a pixel or vertex from processing.
Good for transparency.
