Color Blocks

"hue saturation value" and "hue saturation value alpha"

Returns a colour of the desired hue, saturation, and value.

  • Hue is in a range of (0-360)
  • Saturation is in a range of (0-100)
  • Value is in a range of (0-100)
  • if applicable alpha is in a range of (0-255)

"hue saturation value" and "hue saturation value alpha"

Returns a colour of the desired hue, saturation, and value.
All values are in the range of (0-255).
