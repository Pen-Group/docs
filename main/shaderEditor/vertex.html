Vertex Blocks

These blocks manipulate the corners of a triangle or shape.

"go to [position]", and "go to x: [x] y: [y]"

Makes the current vertex go to the desired position. If the block has a 4-Component vector it will also set depth and w.

"set x to [x]"

Makes the current vertex set it's X value to the desired number.

"set y to [y]"

Makes the current vertex set it's Y value to the desired number.

"set depth to [depth]"

Makes the current vertex set it's depth, or Z, value to the desired number.

"set w to [w]"

Makes the current vertex set it's W, value to the desired number.

"change x by [x]"

Makes the current vertex change it's X value by the desired number.

"change y by [y]"

Makes the current vertex change it's Y value by the desired number.

"change depth by [depth]"

Makes the current vertex change it's depth, or Z, value by the desired number.

"change w by [w]"

Makes the current vertex change it's W value by the desired number.

"make orthographic"

Flattens the current set of vertices but keeps texture warping.

"apply default transforms"

Applies the default vertex transforms that Pen+ applies

"apply default transforms"

Applies the default vertex transforms that Pen+ applies

"input position"

Returns the original input for the vertex's position.

"vertex x"

Returns the current X value of the vertex.

"vertex y"

Returns the current Y value of the vertex.

"vertex depth"

Returns the current depth, or Z, value of the vertex.

"vertex w"

Returns the current W value of the vertex.
